证券

证券交易基础

一，证券交易的概念，基本要素和交易机制

证券交易的概念

已发行的证券在证券市场上买卖的活动
特征有

流动性、收益性和风险性

证券交易的基本要素

证券交易的原则

  1. 公开原则（信息公开）
  2. 公平原则（机会平等）
  3. 公正原则（公正对待）

证券交易的种类

  1. 股票交易
  2. 债券交易
  3. 基金交易
  4. 金融衍生工具交易

证券交易的方式

  1. 现货交易
  2. 远期交易和期货交易
  3. 回购交易
  4. 信用交易

证券交易的有关各方

  1. 证券投资者
  2. 证券公司
  3. 证券交易场所
  4. 证券登记结算机构
  5. 市场中介机构
  6. 市场监管者

证券的交易机制

定期交易和连续交易
指令驱动和报价驱动
证券交易机制目标：

  1. 流动性
  2. 稳定性
  3. 有效性

二，证券交易程序

证券交易程序指投资者通过经纪人在证券交易所买卖证券的交易，要经过开户、委托、竞价成交、结算、过户登记等程序。

证券账户和资金账户

证券账户是证券登记机构为投资者设立
资金账户是证券经纪商为投资者设立的账户

开户的必要性

  1. 了解掌握客户基本情况
  2. 确立委托关系
  3. 防止知情人从中牟利
  4. 方便业务联系

委托

  1. 口头报价
  2. 书面报价
  3. 电脑报价

成交

  • 价格形成方式
    1. 指令驱动方式
    2. 报价驱动方式
  • 竞价方式
    1. 集中竞价
    2. 连续竞价
  • 成立规则

    时间优先，价格优先

  • 竞价结果

结算

买卖双方结清价款和交割证券的过程

证券结算制度

  1. 净额清算制度
  2. 银货对付制度
  3. 共同对手方制度
  4. 分级结算制度

三，委托买卖

委托形式

  • 柜台委托
  • 非柜台委托
  • 委托撤销

委托内容

证券账号、日期、品种、买卖方向、数量、价格、时间、有效期、签名、其他内容

委托价格限制形式

  • 市价委托
  • 限价委托

四，竞价与成交

竞价原则

  • 价格优先
  • 时间优先

  • 行情软件
