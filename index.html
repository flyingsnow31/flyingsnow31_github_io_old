走进证券

2020-02-19
证券

证券交易基础

一，证券交易的概念，基本要素和交易机制

证券交易的概念

已发行的证券在证券市场上买卖的活动
特征有

流动性、收益性和风险性

证券交易的基本要素

证券交易的原则

  1. 公开原则（信息公开）
  2. 公平原则（机会平等）
  3. 公正原则（公正对待）

证券交易的种类

  1. 股票交易
  2. 债券交易
  3. 基金交易
  4. 金融衍生工具交易

证券交易的方式

  1. 现货交易
  2. 远期交易和期货交易
  3. 回购交易
  4. 信用交易

证券交易的有关各方

  1. 证券投资者
  2. 证券公司
  3. 证券交易场所
  4. 证券登记结算机构
  5. 市场中介机构
  6. 市场监管者

证券的交易机制

定期交易和连续交易
指令驱动和报价驱动
证券交易机制目标：

  1. 流动性
  2. 稳定性
  3. 有效性

二，证券交易程序

证券交易程序指投资者通过经纪人在证券交易所买卖证券的交易，要经过开户、委托、竞价成交、结算、过户登记等程序。

证券账户和资金账户

证券账户是证券登记机构为投资者设立
资金账户是证券经纪商为投资者设立的账户

开户的必要性

  1. 了解掌握客户基本情况
  2. 确立委托关系
  3. 防止知情人从中牟利
  4. 方便业务联系

委托

  1. 口头报价
  2. 书面报价
  3. 电脑报价

成交

  • 价格形成方式
    1. 指令驱动方式
    2. 报价驱动方式
  • 竞价方式
    1. 集中竞价
    2. 连续竞价
  • 成立规则

    时间优先，价格优先

  • 竞价结果

结算

买卖双方结清价款和交割证券的过程

证券结算制度

  1. 净额清算制度
  2. 银货对付制度
  3. 共同对手方制度
  4. 分级结算制度

三，委托买卖

委托形式

  • 柜台委托
  • 非柜台委托
  • 委托撤销

委托内容

证券账号、日期、品种、买卖方向、数量、价格、时间、有效期、签名、其他内容

委托价格限制形式

  • 市价委托
  • 限价委托

竞价于成交

竞价原则

  • 价格优先
  • 时间优先

四，实验软件

  • 行情软件
  • 国泰安虚拟交易所

Linux系统设置中文版man手册

2019-12-26
Linux

下载中文包

wget https://src.fedoraproject.org/repo/pkgs/man-pages-zh-CN/manpages-zh-1.5.2.tar.bz2/cab232c7bb49b214c2f7ee44f7f35900/manpages-zh-1.5.2.tar.bz2

安装

tar jxvf  manpages-zh-1.5.2.tar.bz2 #解压
cd manpages-zh-1.5.2 #进入解压文件夹
./configure --disable-zhtw #默认安装 
make && make install #编译安装

新建为cman指令来防止抵消man

cd ~
vim .bash_profile

在其中添加

alias cman='man -M /usr/local/share/man/zh_CN'
source .bash_profile

更新bash_profile 使其生效

数据结构期末考试要点

2019-12-26
考试 / 数据结构

1. 各种题型的答题格式，要回答哪些内容

  • 作业10找重复元素

    给定可能含有重复元素的n个数的数组A[0 .. n-1]。设计一个函数，检查是否存在重复元素。
    (1) 定义函数原型。
    函数的定义
    (2)假设不允许使用额外的空间（可以使用少量的临时变量），并且算法时间复杂度要优于O(n2)，请给出算法的设计思想。
    只写算法思想
    (3)根据算法设计思想，给出伪代码描述的算法步骤，关键之处给出注释，并说明你所设计算法的时间复杂度。
    按照要求，使用注释，时间复杂度要分析

  • 今年(2019)使用新题型

    掌握答题方法，注意答题过程，注重方法分，给出题目要直到从哪里入手，如何展现出来，做题流程。

    2. 考试时算法思想中，要用到一些书上学的算法，要把这些算法写进去吗，怎么把普通二叉树转化成最大值堆

  • 经典的算法不用，可以直接使用相应名字函数来代替。
  • 普通二叉树不能转化成最大值堆。

    3. 考试时哪种问法对应需要写伪代码，思路分析，性能分析等

    算法设计题（无特殊要求），题目中明确告诉的

    4. 期末考试的考察点会侧重期中之后的内容吗

    每一章考一个

    5. 对答题格式有啥要求

    让做啥就做啥

    6. 几大原则

    没上的不考，考的书上不一定有，在视频里找，不要只看书，如果没有笔记，不要ppt。

    7. 复习方法

  • 把学期内所有做过的课程任务看做一遍，所有的考题都练过。
  • 看例题视频(有选择的看)
  • 如果已经看过了，就看知识点，自己讲出来是什么知识点，如果不知道，看课程网站。
  • 做题，自己做一遍，检查对错，时间，每道题要争取在一二十分钟内。
